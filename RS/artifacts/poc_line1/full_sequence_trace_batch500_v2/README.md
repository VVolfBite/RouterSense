This directory contains the 500-prompt Gate1 v2 observational trace outputs.

- `architecture_probe.json`
- `config.json`
- `environment.json`
- `summary.json`

- `trace.jsonl` (~244 MB)

- `gate_weights.pt` (~4.20 GB)
